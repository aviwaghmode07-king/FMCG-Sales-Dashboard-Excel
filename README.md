# FMCG Sales Dashboard (Excel)

## Project Overview
This project is an interactive FMCG Sales Dashboard created using Microsoft Excel.
It analyzes FMCG sales data from 2022 to 2025.

## What is FMCG?
FMCG stands for Fast-Moving Consumer Goods such as food, beverages, personal care, and household products.

## Key Features
- KPI Cards (Total Sales, Total Quantity, Avg Sales, Top Category)
- Sales Trend Analysis (2022–2025)
- State & Channel Performance
- Category-wise Contribution
- Top 10 Products
- Quantity vs Sales Relationship
- Interactive Slicers

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers

## Dataset
- 1000 rows
- Years: 2022–2025
